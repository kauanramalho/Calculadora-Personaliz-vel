# Calculadora Personalizável
Desenvolvi e avaliei uma calculadora web personalizável em um experimento com LLM local, utilizando o Qwen 3.6 27B para gerar o protótipo inicial e o Codex para revisão e testes. Realizei identificação de bugs, refatoração, melhorias de interface, documentação técnica e publicação da aplicação.
Calculadora desenvolvida como experimento de desenvolvimento assistido por IA.

O projeto inicial foi gerado pelo modelo Qwen 3.6 27B e posteriormente
analisado, testado e aprimorado utilizando o Codex.

## Objetivo

Avaliar a capacidade de um LLM local de criar uma aplicação funcional
a partir de um requisito aberto.

## Funcionalidades

- Operações matemáticas básicas
- [Diferencial escolhido pelo modelo]
- Alteração de cores da interface
- Alteração do estilo dos botões
- Menu de personalização
- Interface responsiva

## Minha contribuição

- Definição dos requisitos
- Preparação do prompt

- Identificação e correção de bugs
- Refatoração do código
- Melhoria da experiência do usuário
- Documentação e publicação do projeto
